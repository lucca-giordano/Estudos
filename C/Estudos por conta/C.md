# Começando um projeto

Quando nós criamos um projeto em `C`, nós precisamos definir alguns parametros antes de realmente adcionar algum comando.

```cpp
int main(void){
    //irá executar tudo aqui dentro
}
```

**Não se atente muito a o que `int main(void)` significa**, nós iremos ver isso mais pra frente

Nós podemos importar alguns `header files`, assim adcionando funções novas a linguagem, como a **`stardart io`**, que acaba sendo essencial em muitos casos

```cpp
#include <stdio.h>

int main(void){
    printf("Olá, mundo!");
}
```

`printf()` é uma função que só pode ser obtida caso você importe o `stdio.h` ao seu arquivo, ela irá mostrar a mensagem desejada dentro do terminal, nós iremos usar bastante esse comando para testar algumas hipóteses

nós iremos utilizar bastante a biblioteca do [cs50](https://www.google.com/url?sa=t&rct=j&q=&esrc=s&source=web&cd=&cad=rja&uact=8&ved=2ahUKEwj6woft-M_xAhWNH7kGHSKECCsQFjAAegQIBRAD&url=https%3A%2F%2Fcs50.harvard.edu%2F&usg=AOvVaw29EKs8SwiHspBUajVecd8z), que pode ser baixada através [deste link](https://github.com/cs50/libcs50/tree/main/src). Nessa biblioteca nós encontramos funções como `get_int()`, que serve para armazenar um numero inteiro digitado pelo usuario, e outros tipos de dados que nós iremos tratar daqui a pouco. Caso você não queira baixar essa biblioteca, a propria `stdio.h` tem essa função, e você pode saber mais [clicando aqui](#recebendo-imput-do-usuario).

# Comentarios

Você provavelmente notou o uso de `//` em um exemplo anterios, esses são os comentarios, que são extremamente uteis para fazer anotações dentro do seu código, são pedaços do seu texto que serão completamente ignorados ao rodar o programa, existem duas maneiras de fazer isso:

```cpp
#include <stdio.h>

int main(void){
    printf("1");   // imprime '1' no terminal

    //printf("2");

    printf("3");

    //printf("4");
    
    printf("5");
}
```

Dessa maneira, você está comentando **somente uma linha**, pode ser util quando você quer inutilizar uma linha de código sem apaga-la por completo. Percebe-se que desta maneira, somente o que está a frente do `//` será comentado, assim você pode indicar qual a função da linha de comando após sua declaração.

```cpp
#include <stdio.h>

int main(void){
    printf("1");
    /*
    printf("2");

    printf("3");

    printf("4");
    */
    printf("5");
}
```

E dessa outra maneira, **você esta comentando um trecho do seu código de uma vez só**, você também pode fazer anotações, porém não existe nenhuma maneira de comentar um trecho de seu código e deixar apenas uma ou outra linha funcional, todo o pedaço irá ser ignorado.

# Tipos de variaveis

Assim como em todas as linguagens de programação, em `C` nós precisamos declarar algumas variaveis, e para isso, nós precisamos definir alguns parametros para assim ela exercer alguma função

---

## Tipos de dados

Como `C` é uma linguagem antiga, você precisa ser muito literal, então toda vez que nós declararmos alguma variavel nós precisamos definir qual o **tipo de dado** que ela carrega. Esses tipos são:

>**`int`**, são `numeros inteiros`, carregam no maximo 4 bytes ou 32 bits, sou seja, pode contar até 4 bilhões. Porém esses 4 bilhões são dividos em numeros positivos e negativos, sendo assim sobram 2 bilhões pra cada lado, aparece como `%i` ou `%d` na função `printf()`.
>
>**`char`**, é um tipo de variavel que carrega somente 1 byte, ou seja, `carrega 1 caractere` de acordo com a [tabela ASCII](https://asciichart.com), aparece como `%c` ou `%s` na função `printf()`.
>
>**`float`**, carrega um numero real, ou seja, `um numero decimal`, também armazena 32 bits, e isso pode gerar alguns problemas caso você precise de um numero decimal com bastante precisão, aparece como `%f` na função `printf()`.
>
>**`double`**, tem a mesma função do `float`, porém ao invés de 32 bits, esse tipo de dado carrega até 64 bits, como ele carrega o dobro de dados, ele também acaba sendo `duas vezes mais preciso`, aparece como `%f` na função `printf()`.

Ainda faltam 2 tipos de dados, porém eles **não são nativos da linguagem `C`**, eles estão dentro do arquivo `cs50.h` que foi citado mais cedo e que pode ser facilmente instalado por qualquer um.

>**`bool`**, carrega somente dois valores, `true ou false`. Pode ser substituida por `0 e 1`, porém dessa maneira, fazer alguns testes lógicos se torna uma tarefa mais intuitiva.
>
>**`string`**, é uma coleção de caracteres, pode carregar uma palavra ou uma frase, aparece como `%s` na função `printf()`.

---

## Modificadores

Como você pode perceber, esses tipos de dados mostrados apresentam algumas limitações em alguns aspectos, sendo assim, nós temos os modificadores, que alteram os dados que a variavel pode carregar, assim ela consegue servir melhor para o propósito desejado

>**`unsigned`**, quando usado em `int`, remove a possibilidade de usar numeros negativos, porém abre a possibilidade de usar até o dobro de numeros positivos, quando usado em `char` exerce a mesma função, ao invés de se contar de -128 a 127, você pode contar de 0 a 255
>
>**`long`**, pode ser usado em `int`e em `double`, assim, dobrando o alcançe dos numeros
>
>**`short`**, pode ser usado em `int`, assim, diminuindo o alcançe dos numeros em 2

# Operadores

Assim como qualquer outra linguagem de programação, em `C` nós temos alguns operadores, para fazer contas, testes lógicos e assimilar valores a variaveis.

---

## Operadores matematicos

Aqui sâo os sinais que nós ja conhecemos normalmente para fazer contas basicas

>**`+`** é usado para somar dois ou mais valores
>
>**`-`** é usado para subtrair um valor de outro
>
>**`*`** é usado para multiplicar dois ou mais valores
>
>**`/`** é usado para dividir um valor pelo outro
>
>**`%`** é usado quando o objetivo é obter o que sobrar de uma divisão, retornando o ultimo numero inteiro da conta antes de caucular o primeiro numero decimal

Aqui são os sinais usados para fazer uma soma entre duas variaveis, economizando tempo e tornando o código mais legivel, nós iremos ver isso na pratica daqui a pouco.

>**`++`** soma 1 a variavel, sem precisar colocar nenhum valor adcional
>
>**`+=`** soma um valor desejado
>
>**`-=`** subtrai um valor desejado
>
>**`*=`** multiplica um valor desejado
>
>**`/=`** divide um valor desejado

---

## Funções matematicas

>**`pow()`**, recebe dois argumentos, onde o primeiro é elevado ao segundo

# Declaração e inicialização de variaveis

Nós ja vimos os tipos de dados que as variaveis carregam e os modificadores para ajustar suas funcionalidades, e agora nós iremos ver como elas realmente são declaradas.

Declarar uma variavel é mostrar ao programa que ela existe de alguma maneira, inicializar uma variavel é de fato assimilar um valor a mesma.

Existem duas formas de declarar uma variavel, declarando ela e depois a iniciando com um valor, ou fazer as duas ações na mesma linha. Aqui vamos mostrar as duas formas, onde ambas as variaveis carregam um numero inteiro.

```cpp
int main(void){
    int num1;
    int num2;
    num1 = 0;
    num2 = 4;
}
```

Neste exemplo, você primeiro declara a variavel para depois iniciala com um valor.

```cpp
int main(void){
    int num1 = 0;
    int num2 = 4;
    char nome[4] = "Enzo";
}
```

Neste outro exemplo você declara a variavel junto de seu valor atribuido, você pode perceber duas coisas diferentes, a variavel do tipo `char` está carregando mais de um caractere, e depois do nome dela é usado um `[]`. Estes colchetes são usados para indicar que a variavel vai carregar mais de um valor, assim formando esta palavra, e o numero dentro desses colchetes indica quantos caracteres essa variavel irá carregar.

```cpp
int main(void){
    int num1;
    int num2;
    char frase[14];
    num1 = 0;
    num2 = 4;
    frase = "Ola, tudo bem?";
}
```

Aqui nós também precisamos indicar qual o tamanho da string na variavel `frase`, mesmo antes de atribuir qualquer valor a ela.

```cpp
int main(void){
    int altura, largura;
    float raiz1, raiz2, raiz3;

    altura = 4;
    largura = 3;

    raiz1 = 0.23;
    raiz2 = 0.234;
    raiz3 = 0.2345;
}
```

Aqui você declarou mais de uma variavel na mesma linha, e atribuiu seus valores nas linhas seguintes.

Todos esses exemplos estão corretos e funcionam da mesma maneira, isso depende da maneira que você acha mais facil e mais organizada de fazer isso.

## Declarando constantes

Constantes são parecidas com as variaveis, porém seu valor não pode ser alterado de nenhuma maneira. Recebem os mesmos tipos e funções das variaveis, a unica diferença é que elas não fazem parte de operações matematicas ou são alteradas de alguma maneira.

```cpp
int main(void){
    const int num1 = 0;
    int const num2 = 4;
}
```

Percebe-se que antes ou depois de declarar o tipo, a palavra `const`, ambas maneiras estão corretas e irão ter o mesmo efeito.

# Recebendo imput do usuario

Dentro do arquivo `stdio.h` nós temos uma função que recebe o imput do usuario, porém nós iremos usar a biblioteca do `cs50` pela sua simplicidade. 

Dentro do `stdio.h` nós temos duas maneiras de fazer isso, usando o `scanf()` e o `fgets()`.

No `scanf()`, você precisa de dois argumentos, o tipo de dado que irá detectar, e a variavel na qual esse dado irá ser armazenado

```cpp
#include <stdio.h>

int main(void){
    int num;
    printf("Selecione um numero: ");
    scanf("%i", &num);
    printf("O numero é: %i", num);
}
```

Vale lembrar que a função `scanf()` possui algumas limitações, para escanear um numero decimal do tipo `double`, o primeiro argumento precisa ser `%lf%`, e para escanear alguma frase, ele irá somente armazenar as letras até o primeiro espaço em branco.

Já no `fgets()`, você irá precisar de três argumentos, o nome da variavel, o tamanho da string que você irá armazenar, e o local onde a informação irá ser retirada, porém ele só é aplicavel a variaveis do tipo `char`:

```cpp
#include <stdio.h>

int main(void){
    char nome[20];
    printf("Digite seu nome: ");
    fgets(nome, 20, stdin);
    printf("O nome é: %s", nome);
}
```

Como nós vimos antes, ele primeiro exige o nome da variavel, depois a quantidade de caracteres, e depois o local de onde esta informação irá ser retirada. O ponto é, esses `stdin` é uma nova expressão, e ela significa `standart`, ou seja, `padrão`. Dessa maneira, ele irá decetar a informação pelo caminho padrão, que nesse caso é o terminal onde as informações serão digitadas pelo usuario.


Daqui pra frente nós iremos usar as funções inclusas no `cs50.h` por serem mais simples, mas você pode somente substituir os comandos se desejar.

# Testes lógicos

Você ja ouviu essa expressão diversas vezes dentro desse texto, e agora nós iremos nos aprofundar no que são estes testes lógidos e como são feitos.

Em toda linguagem de programação, todos os programas são feitos com perguntas de sim ou não, e com os caminhos posiveis para a resposta obitida em tais perguntas. Estes são os testes lógicos, você faz uma comparação entre dois ou mais valores, existem 3 maneiras de realizar esta tarefa:

---

## if, else, else if

A maneira mais comum de realizar estes testes é usando as funçoes `if` e `else`, aqui está um exemplo:

```Cpp
#include <stdio.h>
#include <cs50.h>

int main(void){
    int num = get_int("Insira um numero: ");
    if (num > 10){
        printf("Seu numero é maior que 10");
    }
    else{
        printf("Seu numero nao é maior que 10");
    }
}

```

Como você pode perceber, nós perguntamos ao usuario algum numero qualquer, dentro do `if`, ele testa se o numero atribuido a variavel `num` (que foi digitado pelo usuario) é maior que 10, se for, ele irá executar tal ação.

Logo em seguida você vê o `else`, ele sempre fica em ultimo nas ramificações do seu teste lógico, e o que estiver la dentro só será executado se o resultado de todos os outros testes for falso

```cpp
#include <stdio.h>
#include <cs50.h>

int main(void){
    int num = get_int("Insira um numero: ");
    if (num > 10){
        printf("Seu numero é maior que 10");
    }
    else if (num == 10){
        printf("Seu numero é igual a 10");
    }
    else if (num < 10){
        printf("Seu numero é menor que 10");
    }
}
```

Aqui nós usamos o `else if`, ele é usado quando seu teste lógico está dentro do mesmo braço, porém é necessario executar mais de um teste para se obter o resultado final.

Nota-se que não foi ultilizado o `else`, porém neste caso, como nós sabemos que só existem 3 possibilidades (ou o numero é maior, ou é menor ou é igual), nós poderiamos usar somente um `else` no final sem ter nenhum teste. Isso vai de cada um, ambas as maneiras estão corretas.

---

## switch, case

Aqui nós iremos ver alguns comandos novos, o `switch`, o `case`, o `break` e o `default`

O `switch` é usado quando nós queremos avaliar um valor de uma variavel, e o `case` é usado para testar o valor da variavel.

O `break` é usado para terminar as ações dentro do `case`, e o `default` é o que será executado caso o valor não corresponda com nenhum avaliado dentro do `case`.

Neste pequeno programa, nós iremos pegar um numero inteiro do usuario, e testar se ele está entre 1 e 3.

```cpp
#include <stdio.h>
#include <cs50.h>

int main(void){
    int num = get_int("Digite um numero: ");

    switch (num){  //seleciona a variavel a ser avaliada
        case 1 :
            printf("O numero é 1");
            break;  //termina a execção

        case 2 :
            printf("O numero é 2");
            break;

        case 3 :
            printf("O numero é 3");
            break;

        case 4 :
            printf("O numero é 4");
            break;

        default : // caso o valor não corresponda a nenhum teste
            printf("Seu numero é: %i, não está entre 1 ou 4.", num);
    }
}
```

Esse código, apesar de maior, é bem simples. Ele testa se o numero corresonde aos testados pelo `case`, e se não for, as ações dentro do `default` irão ser executadas.

Perceba também que não é necessario usar `break` para encerrar a execução do `default`.

O interessante de usar `switch` e `case` é que ele não trabalha com operadores relacionais, ou seja, ele só executa se o valor for aquele e somente aquele, ele não compara com outro valor. É muito usado para fazer menus, já que dentro de um menu ja existem valores fixados anteriormente.

Se este mesmo código fosse escrito usando `if`, `else`, e `else if`, ficaria desta maneira:

```cpp
#include <stdio.h>
#include <cs50.h>

int main(void){
    int num = get_int("Digite um numero: ");

    if (num == 1){
            printf("O numero é 1");
    }
    else if (num == 2){
            printf("O numero é 2");        
    }
    else if (num == 3){
            printf("O numero é 3");   
    }
    else if (num == 4){
        printf("O numero é igual a 4");
    }
    else{
        printf("Seu numero é: %i, não está entre 1 ou 4.", num);
    }
}
```

Ambos funcionam **exatamente da mesma maneira**, e **ambas estão corretas**, porém tendo essa visualização das duas maneiras fica mais simples de entender qual a função de cada comando dentro do `switch`.

---

## Operação ternária

Esse tipo de teste é um pouco mais especifico, porém ocupa muito menos espaço no seu código, e pode te ajudar a se organizar e tornar o código mais legivel. O nome desse tipo de teste é `operação ternária`, recebe esse nome pois necessita de 3 expressões para ser realziada.

```cpp
#include <stdio.h>
#include <cs50.h>

int main(void){
    int num = get_int("Insira um numero: ");
    (num > 10) ? printf("É maior que 10") : printf("É menor que 10");

}
```

Isso parece ser muito complexo, mas é bem mais simples do que parece. Essa linha primeiro faz um teste lógico, se o valor vor verdadeiro, ele irá executar a primeira ação, se o valor for falso, ele irá executar a segunda ação.

Escrevendo usando `else if` fica dessa maneira:

```cpp
#include <stdio.h>
#include <cs50.h>

int main(void){
    int num = get_int("Insira um numero: ");

    if (num > 10){
        printf("É maior que 10");
    }
    else{
        printf("É menor que 10");
    }
}
```

Ambas esão corretas, ambas executam a mesma função, porém uma delas ocupa muito menos espaço no seu código e fica muito mais organizada, a outra talvez seja mais legivel, porém vai de cada um.

# Loops

Loops servem para executar um ou mais comandos repetidas vezes, sem precisar escrever a mesma linha mais de uma vez. Existem algumas maneiras de fazer:

---

## while

While é usado quando nós não sabemos quando o código vai parar de repetir, ele está ligado a algum teste lógico, e só será executado se o valor for verdadeiro. Existem 3 maneiras de usar o `while`:

---

### **while()**

A maneira mais comum é cusando junto de um teste lógico, ou seja, antes de executar a ação ele checa se o teste ainda está retornando como verdadeiro.

```cpp
#include <stdio.h>
#include <cs50.h>

int main(void){
    int num = get_int("Digite um numero: ");
    while(num != 0){
        printf("falta %i vezes para acabar\n", num);
        num--;
    }
}
```

Neste exemplo, o programa pede um numero ao usuario, subtrai `1` da variavel ,e enquanto o numero for diferente de 0, ele irá imprimir quantas vezes faltam para acabar o loop.

---

### **while(true)**

É um loop infinito, não irá ser interrompido até o comando `break` ser iniciado.

```cpp
#include <stdio.h>

int main(void){
    int num = 0;
    while(true){
        printf("%i\n", num);
        num++;
    }
}
```

Aqui a variavel `num` irá ser imprimida no terminal infinitamente, perceba que toda vez que o loop reinicia, então o numero irá sempre aumentar em `1`.

```cpp
#include <stdio.h>
#include <cs50.h>

int main(void){
    while(true){
        int num = get_int("Digite um numero: ");
        if (num == 999){
            break;
        }
    }
}
```

Aqui nós fizemos a aplicação do `break`, onde o programa irá perguntar ao usuario para ele digitar um numero infinitamente, porém se o numero for igual a `999`, o `break` será acionado e o loop será interrompido.

---

### **do{}while()**

Dessa maneira, o comando será executado pelo menos uma vez antes do teste lógico ser realizado, como o `while` normalmente é usado quando você não tem certeza se o comando vai ser executado ou quantas vezes ele vai ser executado, dessa maneira você garante que o código irá ser executado independentemente do resultado do teste, ele só não irá se manter em loop.

```cpp
#include <stdio.h>
#include <cs50.h>

int main(void){
    int num1, num2; 
    char resp;
    do{
        num1 = get_int("Digite o primeiro numero: ");
        num2 = get_int("Digite o segundo numero: ");
        printf("A soma de %i e %i é %i", num1, num2, num1 + num2);
        resp = get_char("Quer continuar? [S/N] ");
    }while(resp == 's');
}
```

Aqui você pode perceber que o teste lógico só retorna verdadeiro se `resp` for igual a `'s'`, porém também é possivel notar que a varivel `resp` é iniciada sem nenhum valor atribuido, e mesmo assim o programa funciona normalmente, e isso é devido ao `do{}while()`

---

## for

O `for` é um pouco mais complexo que o `while`, ele é usado quando você precisa executar uma ação em uma quantidade determinada de vezes, ele recebe uma certa construção junto de uma variavel de controle, assim voce executar as mesmas funções de um `while` economizando menos linhas e deixando seu código mais organizado.

```cpp
#include <stdio.h>
#include <cs50.h>

/*
A construção do for é feita dessa maneira:

for(variavel de controle; teste lógico; controle da variavel){
    código a ser executado
}

Agora vamos ver na pratica:
*/

int main(void){
    for(int i = 10; i > 0; i--){
        printf("posicao %i\n", i)
    }
}
```

Aqui você inicia uma variavel, nesse caso, a variavel `i`. Logo em seguida, você faz um teste lógico para saber se o loop ainda irá se repetir, e logo em seguida você controla o valor da variavel. Caso o teste lógico retornar como verdadeiro, ele irá executar a ação. Note que a variavel `i` pode ser usada normalmente, porém sempre lembrando que ela irá ter o valor determinado pelo controle da variavel a cada vez que o loop roda.

# Arrays

Nossos códigos as vezes ficam enormes, centenas, milhares e até dezenas de milhares de linhas de código. Isso pode ficar um pouco confuso quando se trata de organizar dados, ainda mais quando esses dados estão relacionados de alguma maneira. Para isso, nós podemos usar as `arrays`, as arrays são um tipo de declarar uma variavel.

Vamos olhar este exemplo:

```cpp
#include <stdio.h>

int main(void){
    int num1 = 0;
    int num2 = 12;
    int num3 = 43;

    printf("primeiro: %i, segundo: %i, terceiro: %i", num1, num2, num3)
}
```

Você pode perceber que nós temos 3 variaveis, todas elas tem o mesmo propósito, porém ainda assim elas são diferentes e ocupam mais espaço dentro do seu código. Assim, nós podemos fazer desta maneira: 

```cpp
#include <stdio.h>

int main(void){
    int num[3] = {0, 12, 43};
    printf("primeiro: %i, segundo: %i, terceiro: %i", num[0], num[1], num[2]);
}
```

Nós vemos algumas coisas diferentes aqui, a variavel está sendo declarada com `[3]` do lado, isto significa que a variavel **é uma array** e que esta array irá carregar 3 dados diferentes, **o numero dentro dos colchetes indica `a quantidade de dados total que a variavel irá carregar`**. Porém, quando nós aplicamos ela dentro do código, **a primeira casa é sempre indicada como a `casa 0`, e as outras só seguem a sequencia normalmente**, dentro das chaves, os numeros são alocados as suas respectivas casas dentro da array. Ela também pode ser declarada assim: 

```cpp
#include <stdio.h>

int main(void){
    int num[3];
    num[0] = 0;
    num[1] = 12;
    num[2] = 43;
    printf("primeiro: %i, segundo: %i, terceiro: %i", num[0], num[1], num[2]);
}
```

Ambas funcionam da mesma maneira e irão executar as mesmas ações, porém depende de como você acha mais organizado.

---

## Strings

Dentro da linguagem `C`, você pode ter percebido que nós não temos o tipo de dado `string` de maneira nativa, porém nós temos outra maneira de definir frases, usando as `arrays.

```cpp
int main(void){
    char frase[11] = "Ola, mundo!"
}
```

Aqui nós armazenamos o valor `"Ola, mundo!"` dentro de uma variavel do tipo `char`, você pode fazer isso, esta é uma string, porém você precisa indicar quantos caracteres a frase irá ter dentro dos colchetes, o interessante de fazer desta forma, é que nós podemos manipular cada caractere de maneira individual, tendo em vista que ele assimila cada caractere a uma casa da array de maneira automatica.

# Structs

`Structs` são outra maneira de organizar dados, como nós vimos nas arrays, elas podem carregar diversos valores **desde que todos sejam do mesmo tipo de dado**, dentro das `structs`, isso não importa. Vamos ver como elas são declaradas:

```cpp
struct Estudante{
    int idade;
    char nome[50];
    double nota;
};
```

Certo, aqui nós temos a `struct Estudante` declarada com sucesso, ela recebe 3 dados, um inteiro chamado `idade`, uma string chamada `nome`, e um double chamado `nota`. Repare que nesse caso as chaves também são seguidas por `";"`.

Certo, nós temos as variaveis declardas, porém nós não as iniciamos com nenhum valor, vamos ver como isso é feito:

```cpp
#include <stdio.h>
#include <string.h>

struct Estudante{
    int idade;
    char nome[50];
    double nota;
};

int main(void){
    struct Estudante aluno1;
    aluno1.idade = 13;
    aluno1.nota = 6.7;
    strcpy(aluno1.nome, "José");
}
```

Certo, isso parece muito confuso, porém não é, Como nós temos o `struct Estudante`, nós podemos criar galhos dele, que recebem todos os parametros declarados dentro da struct, porém podem ser separados com nomes diferentes, neste caso, nosso galho se chama `aluno1`.

Dentro deste galho, nós inicializamos a sua idade usando `aluno1.idade`, e assim por diante. 

Porém temos algo diferente aqui, esta função `strcpy()` não é incluida por padrão na linguagem e não está dentro da biblioteca `stdio`, por conta disso, nós precisamos importar a `string.h`.

Por padrão, nós não conseguiriamos fazer por exmplo `aluno1.nome = "José"`, por conta disso nós usamos essa função, ela necessita de dois argumentos, a variavel destino, e a string de origem, que pode ser uma variavel ou escrita dessa maneira ja vista. 

Assim, nós podemos criar diversos galhos que recebem os mesmos parametros porém com valores diferentes:

```cpp
#include <stdio.h>
#include <string.h>

struct Estudante{
    int idade;
    char nome[50];
    double nota;
};

int main(void){
    struct Estudante aluno1;
    struct Estudante aluno2;

    aluno1.idade = 13;
    aluno2.idade = 14;
    aluno1.nota = 6.7;
    aluno2.nota = 7.6;
    strcpy(aluno1.nome, "José");
    strcpy(aluno2.nome, "Maria");
}
```

Aqui nós temos dois desses galhos, `aluno1` e `aluno2`, ambos recebem as mesmas variavels, porém os valores são diferentes um do outro. Para imprimir estes valores no terminal, nós fazemos desta maneira:

```cpp
#include <stdio.h>
#include <string.h>

struct Estudante{
    int idade;
    char nome[50];
    double nota;
};

int main(void){
    struct Estudante aluno1;
    aluno1.idade = 13;
    aluno1.nota = 6.7;
    strcpy(aluno1.nome, "José");

    printf("Nome: %s. Idade: %i, Nota: %lf", aluno1.nome, aluno1.idade, aluno1.nota);
}
```

Ou seja, para inicializar valores e usar esses valores nós usamos esta construção: `nome_do_galho.dado_a_ser_utilizado`

# Funções

Esta é uma parte **muito importante**, porém um pouco mais complicada e bem extensa. Funções podem ser vistas como blocos escuros no seu código, onde ela **recebe um ou mais dados**, faz os cauclos necessarios, e **retorna o valor esperado**.

As funções recebem alguns parametros em sua declaração, elas são construidas desta meneira:

```cpp
int add(int x, int y);  
```

O `int` é o tipo de dado que a função irá retornar.

O `add` é o nome da função, que normalmente é sempre relacionado com o que esta função vai fazer, para facilitar a organização de seu código.

E o `int x, int y` são os argumentos que a variavel vai receber para executar as ações.

Vamos abrir um pequeno projeto para entender como as funções funcionam: 

```cpp
#include <stdio.h>
#include <cs50.h>

// protótipos
int soma(int a, int b);

// função main
int main(void){

    // recebe um input do usuario
    int x = get_int("Insira o primeiro numero: ");
    int y = get_int("Insira o segundo numero: ");

    // a variavel z é o valor que a função soma irá retornar
    int z = soma(x, y);

    //imprime o resultado
    printf("A soma de %i e %i é %i", x, y, z);
}

// função soma      
int soma(int a, int b){

    // o valor de c é a soma dos dois argumentos inseridos
    int c = a + b;

    // o valor retornado é C
    return c;
}
```

Calma, isso parece muito confuso, porém é bem simples. Como o computador sempre lê o código de cima pra baixo, nós precisamos colocar os `protótipos`, como nós chamamos a função antes de a declarar, nós precisamos indicar ao programa que esta função vai existir em algum momento.

A função `main` é um padrão para inicializar o código, ela é um ponto de partida para se programa ser lido da maneira correta. Apesar de parecer ser uma função qualquer, ela é necessaria e não pode ter outro nome, caso você tente, você irá ver um erro.

A função `soma` é a quel faz todo o trabalho com as variaveis passadas dentro dela na função `main`.

Você pode perceber que dentro da função `main`, como ela não recebe argumentos, se usa o `void`, que indica ao programa que a função não irá receber nenhum argumento para se trabalhar.

Existe algo que é importante de ser notado, a função só consegue trabalhar se você fornecer argumentos o suficiente, caso você tenha construido ela para receber 3 argumentos, você precisa lhe dar 3 para trabalhar, você irá receber um erro caso use mais ou menos argumentos.

# Escopo de variavel

Apesar do nome ser confuso, isso é extremamente simples. Existem dois tipos de escopo de variaveis, a `local` e a `global`. 

As `variaveis locais` são declaradas dentro da função que irá ser utilizada, porém não podem ser usada em nenhuma outra parte do seu código. 

As `variaveis globais` são declaradas fora das funções, porém podem ser usadas em qualquer lugar do seu código.

Vamos usar esse código que ja foi compreendido anteriormente como exemplo

```cpp
#include <stdio.h>
#include <cs50.h>

int soma(int a, int b);

int num;

int main(void){
    int x = get_int("Insira o primeiro numero: ");
    int y = get_int("Insira o segundo numero: ");
    num = 10;
    int z = soma(x, y);
    printf("A soma de %i e %i é %i", x, y, z);
}

int soma(int a, int b){
    int c = a + b + num;
    return c;
}
```

A variavel `num` está sendo declarada fora de qualquer função, sendo assim ela se torna global, pode ser usada em qualquer lugar, Nós a declaramos do lado de fora, inicializamos dentro da função `main`, e só realmente a usamos para alguma coisa dentro da função `soma`. 

As variaveis `x`, `y` e `z` fazem parte da função `main`, e não podem ser usadas dentro da função `soma`, assim como as variaveis `a`, `b` e `c` fazem parte da função `soma`, e não pode ser usada dentro da função `main`. 

Isso pode parecer inutil de certa forma, e apesar de poder causar alguns problemas quando nós nomeamos nossas variaveis, é util para fazer comunicações entre as funções sem precisar criar novas variaveis e novas funções para retornar esse valor.
